## For development
Clone the project:

```bash
git clone https://github.com/soponkntg/find-my-mines
```

First, install all dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
